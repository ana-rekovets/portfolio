Causal Inference · Field Experiment · LMU Munich · Mar 2026

When Is TSLS Actually LATE? Evidence from Project STAR

Replication and extension of Krueger (1999). Applied DDML and IPSW estimators to test whether TSLS retains a valid Local Average Treatment Effect interpretation under covariate inclusion, following Blandhol et al. (2022). The rich covariates condition fails in all grade cohorts — by Grade 3, specification bias accounts for 32.6% of the TSLS estimate, exceeding the selection bias the IV strategy was designed to eliminate.

Supermarket Chain — Relational Database

Full design and implementation of a normalized relational database for a multi-country supermarket chain in MySQL. Covers the complete pipeline from EER diagram to 3NF schema to 15 analytical queries spanning joins, correlated subqueries, and relational division.

Models customers, transactions, products, store inventory, payment methods, and a loyalty points program with full earn/redeem audit trail. Historical pricing stored at transaction level — past receipts remain accurate when list prices change.

  • POINT_OF_SALE 3 stores · Switzerland, Italy, Germany
  • CUSTOMER + LOYALTY_CARD 1:1 enforced via UNIQUE(customer_id)
  • TRANSACTION Composite FK → PAYMENT_METHOD(point_id, method)
  • PRODUCT_IN_TRANSACTION Line items with historical unit_price at purchase time
  • POINTS_MOVEMENT Full earn/redeem audit trail · nullable transaction_id

-- Q14: Relational division
-- stores accepting ALL payment methods

SELECT pos.point_id,
       pos.name, pos.city
FROM POINT_OF_SALE pos
WHERE NOT EXISTS (
  SELECT 1
  FROM (
    SELECT DISTINCT payment_method
    FROM PAYMENT_METHOD
  ) allm
  WHERE NOT EXISTS (
    SELECT 1
    FROM PAYMENT_METHOD pm
    WHERE pm.point_id = pos.point_id
      AND pm.payment_method
        = allm.payment_method
  )
)
ORDER BY pos.point_id;
